    Terminology - Loops 

    Last Updated: 2024-04-03

    do-while-loop 

    • A Do-while-loop is a post-test loop. Do these-lines-of-code while this-is-true. Do-While Loop - (a post-test loop)

    while-loop 

    • A While-loop is a pre-test loop. The contents of the loop may not be executed if the condition is initially false. While-Loop - (a pre-test loop)

    for-loop 

    • A For-loop is a pre-test loop consisting of 3 parts: initialize the counter; test the counter; increment the counter. For-Loop - (a pre-test loop)

    nested loops 

    • Nested loops are loops that can be nested inside each other. Commonly used with two-dimensional arrays. Nested Loops

    pre-test loop 

    • A loop where the test occurs before the code to be executed. The code inside the loop may not be executed at all. While-Loop - (a pre-test loop) For-Loop - (a pre-test loop)

    post-test loop 

    • A post-test loop is a loop where the code inside the loop is always executed at least once because the test occurs after the code in the loop is excuted. Do-While Loop - (a post-test loop)](@/loops/loops-do-while-loop-post-test-loop.md)

    sentinel 

    • A sentinel is an inputted value that tells the program to stop and do something else.

    infinite loop 

    • An infinite loop is a loop that never ends. This is not good.

    initialization 

    • Initialization is setting a variable or the elements of an array to a value.

    counter 

    • A counter is a variable used to count the number of occurrences of an item. Example: counter++

    increment 

    • Increment is to increase the value of a variable. Example: x++

    decrement 

    • Decrement is to reduce. Example of x minus one: x--

    wantsToContinue Loop 

    • A wantsToContinue loop is a standard loop that asks the user if they want to continue. Commonly wants a Y or N response. Also called a do-it-again-loop. wantsToContinue Loop

    game loop 

    • A game loop is a loop frequently used in games. Game Loop

    counting loop 

    • A counting loop is a loop used to count the number of items either being entered or already in an array. Counting Loop - (a pre-test loop and a post-test loop)
